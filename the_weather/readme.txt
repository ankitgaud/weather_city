Installation :-
		1. Install Python 3.7.4
		2. Install python packages - Django, request, selparser
			and someother as project requirement.





Weather information -:

		In this project, I have used "openweathermap" api.
		This api provides a lot of current information of 
		every city.
		It provide output with city name as parameter.


Django -:
		
		I have developed this project on python-Django.
		In this, We can use dashboard after login and 
		You can register yourself and good use of django 
		middlewares according to project requirement.

		In this, I have used default database sqllite.
		For user login and registration, I have not used 
		builtin form. I have created myself.


For frontend -: 
		I have used HTML, CSS and bootstrap.
